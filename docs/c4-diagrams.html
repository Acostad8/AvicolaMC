Sistema Avícola MC

Diagramas C4 – Nivel 1 (Contexto del Sistema) y Nivel 2 (Contenedores)

🎯

Propósito

Gestión integral de granjas avícolas: lotes, producción, mortalidad, tratamientos e inventario.

👤

Administrador

Gestiona usuarios, proveedores, empleados, configuración y tiene acceso total al sistema.

🐔

Encargado de Galpón

Registra producción diaria, mortalidad, tratamientos e insumos para sus galpones asignados.

☁️

Supabase (BaaS)

Provee autenticación JWT, base de datos PostgreSQL y funciones serverless (Edge Functions).

⚛️

React SPA

React 19 + Vite 8. Enrutamiento con React Router v7. Estado del servidor con TanStack Query v5.

🔐

Supabase Auth

Autenticación por email/contraseña. Tokens JWT. Recuperación de contraseña por correo.

🗄️

PostgreSQL + RLS

15 tablas operacionales. Row Level Security: admins ven todo; encargados solo sus galpones.

⚡

Edge Functions

Función create-user en Deno/TypeScript. Crea usuarios con verificación de permisos de admin.
